CSS Boxing model

Container 1

With 'box-sizing' property not defined (on each div)

Total div size = width + height + padding + border

Additional note: there´s a collapse of margins between each div,

to solve it, check Container 4

width: 250px; height: 250px;

padding: 5px;

border: 4px solid black

margin: 8px;

box-sizing no defined

width: 250px; height: 250px;

padding : 5px;

border: 8px solid orange;

margin: 8px;

box-sizing no defined

Container 2

box-sizing: border-box;

With 'box-sizing: border-box;' (on each div)

the size of width and height, is respected without the sizes of padding, border or margin

Total div size = 250px x 250px
