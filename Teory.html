Render Engine on Browsers

The main function of a browser engine is to transform the HTML documents and other resources of a web page into an interactive visual representation on the user's device (this function is called rendering).

Selectors

Basics

type: div {...}
class: .elemento {...}
ID: #id-del-elemento
attributte: a[href="..."]{...}
universal: *{...}

Combiners

Descendant: div p
direct child: div > p
adyacent element: div + p
General of brothers: div ~ p

Pseudoclasses

Pseudoclasses allow us to get to those actions that the user does.
:activate
:focus
:hover
:nth-child(n)

Pseudo elements

Pseudo-elements allow us to access HTML elements that are not accessible with the selectors already seen. ::after
::before
::first-letter
::placeholder
The pseudo-class is written with : and the pseudo-elements are written with ::
